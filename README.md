# 📥 Essential Software Downloads

## Code Editors / IDEs
- [Visual Studio Code](https://code.visualstudio.com/) — lightweight, most popular editor
- [PyCharm Community](https://www.jetbrains.com/pycharm/download/) — Python-focused IDE
- [Android Studio](https://developer.android.com/studio) — for Android/mobile dev

## Local Dev Environments (PHP/MySQL)
- [XAMPP](https://www.apachefriends.org/) — Apache, MySQL, PHP bundled
- [Laravel Herd](https://herd.laravel.com/) — lightweight Laravel dev environment (Windows/Mac)

## Programming Languages / Runtimes
- [Python](https://www.python.org/downloads/)
- [Node.js](https://nodejs.org/) — includes npm
- [PHP](https://windows.php.net/download/) — Windows builds
- [Git](https://git-scm.com/downloads)

## Databases / DB Tools
- [MySQL Community Server](https://dev.mysql.com/downloads/mysql/)
- [MySQL Workbench](https://dev.mysql.com/downloads/workbench/)
- [DBeaver](https://dbeaver.io/download/) — free universal DB client
- [Postman](https://www.postman.com/downloads/) — API testing

## Version Control / GitHub Tools
- [Git for Windows](https://git-scm.com/download/win) — includes Git Bash
- [GitHub Desktop](https://desktop.github.com/) — GUI alternative to CLI

## Browsers (for testing)
- [Google Chrome](https://www.google.com/chrome/)
- [Mozilla Firefox](https://www.mozilla.org/firefox/)

## Office / Productivity
- [LibreOffice](https://www.libreoffice.org/download/download/) — free MS Office alternative
- [7-Zip](https://www.7-zip.org/) — file compression

## Networking / Security Tools
- [Wireshark](https://www.wireshark.org/download.html) — packet analysis
- [Npcap](https://npcap.com/#download) — required for Wireshark on Windows

## Design / Diagramming
- [Figma](https://www.figma.com/downloads/) — UI/UX design
- [draw.io / diagrams.net](https://www.diagrams.net/) — free diagramming tool

## Communication
- [Discord](https://discord.com/download)
- [WhatsApp Desktop](https://www.whatsapp.com/download)
